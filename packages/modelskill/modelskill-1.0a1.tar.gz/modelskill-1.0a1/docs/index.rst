ModelSkill: Assess the skill of your MIKE FM model
==================================================

Compare results from MIKE FM simulations with observations. `ModelSkill <https://github.com/DHI/modelskill>`_ would like to be your companion during the different phases of a MIKE FM modelling workflow.

This package is simple to install and deploy operationally and is accessible to everyone as open-source.

Start by reading the `Getting started guide <getting_started.html>`_ and browsing the `example notebooks <https://nbviewer.jupyter.org/github/DHI/modelskill/tree/main/notebooks/>`_.
